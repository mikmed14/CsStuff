How to play the scripts:

1) Download and install CSound here https://sourceforge.net/projects/csound/files/csound6/Csound6.06/

2)Download the script

3)Open a terminal/msDos and move into the folder where you saved the script

4)Write on the command line the following line:
csound <name_of_the_script>

5)Enjoy the listening
